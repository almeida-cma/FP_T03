Gerando Números Aleatórios
Biblioteca Util: Para usar funções aleatórias, é necessário importar a biblioteca Util com inclua biblioteca Util e usar Util.sorteia(min, max)!

Números Aleatórios com Util.sorteia()

A função Util.sorteia(menor, maior) gera números inteiros aleatórios dentro do intervalo especificado.
programa { inclua biblioteca Util funcao inicio() { inteiro numero_aleatorio numero_aleatorio = Util.sorteia(1, 100) escreva("Número aleatório gerado: ", numero_aleatorio) } }

Util.aguarde() e Util.tempo_decorrido()

A função Util.aguarde(milissegundos) pausa o programa. Util.tempo_decorrido() retorna o tempo desde o início.
programa { inclua biblioteca Util funcao inicio() { inteiro inicio, fim inicio = Util.tempo_decorrido() escreva("Aguardando 2 segundos...\n") Util.aguarde(2000) fim = Util.tempo_decorrido() escreva("Tempo decorrido: ", fim - inicio, " ms") } }
Laços de Repetição: ENQUANTO, FAÇA-ENQUANTO e PARA

Laço ENQUANTO (Pré-Testado)

Executa enquanto a condição for verdadeira. Testa ANTES de executar.
programa { funcao inicio() { inteiro contador = 1 enquanto (contador <= 5) { escreva("Repetição número: ", contador, "\n") contador = contador + 1 } escreva("Fim do laço!") } }

Laço FAÇA-ENQUANTO (Pós-Testado)

Executa pelo menos uma vez, repete enquanto a condição for verdadeira. Testa DEPOIS de executar.
programa { funcao inicio() { inteiro contador = 1 faca { escreva("Repetição número: ", contador, "\n") contador = contador + 1 } enquanto (contador <= 5) escreva("Fim do laço!") } }

Laço PARA (Com Variável de Controle)

Ideal quando sabemos exatamente quantas repetições serão necessárias. Possui inicialização, condição e incremento.
programa { funcao inicio() { para (inteiro i = 1; i <= 5; i++) { escreva("Repetição número: ", i, "\n") } escreva("Fim do laço!") } }

Exercício 2: Gerador de Loteria

Desafio: Gere 6 números aleatórios entre 1 e 60 para jogar na loteria! Use laços PARA e FAÇA-ENQUANTO.
programa { inclua biblioteca Util funcao inicio() { inteiro numeros[6] inteiro i, j, temp logico repetido escreva("=== NÚMEROS DA LOTERIA ===\n") escreva("Gerando 6 números entre 1 e 60...\n\n") // Gera 6 números únicos para (i = 0; i < 6; i++) { faca { numeros[i] = Util.sorteia(1, 60) repetido = falso // Verifica se o número já foi sorteado para (j = 0; j < i; j++) { se (numeros[i] == numeros[j]) { repetido = verdadeiro } } } enquanto (repetido == verdadeiro) } // Ordena os números (ordem crescente) para (i = 0; i < 5; i++) { para (j = i + 1; j < 6; j++) { se (numeros[i] > numeros[j]) { temp = numeros[i] numeros[i] = numeros[j] numeros[j] = temp } } } escreva("Seus números da sorte: ") para (i = 0; i < 6; i++) { escreva(numeros[i], " ") } escreva("\n\nBoa sorte! 🍀") } }

Exercício 3: Jogo dos Palpites

Desafio: Dois jogadores fazem palpites. Quem chegar mais perto do número aleatório vence!
programa { inclua biblioteca Util funcao inicio() { inteiro numero_secreto, palpite1, palpite2 inteiro diferenca1, diferenca2 escreva("=== JOGO DOS PALPITES ===\n") numero_secreto = Util.sorteia(0, 100) escreva("Jogador 1, digite seu palpite (0 a 100): ") leia(palpite1) escreva("Jogador 2, digite seu palpite (0 a 100): ") leia(palpite2) escreva("\n--- RESULTADO ---\n") escreva("Número secreto: ", numero_secreto, "\n") escreva("Palpite do Jogador 1: ", palpite1, "\n") escreva("Palpite do Jogador 2: ", palpite2, "\n\n") // Calcula as diferenças (valor absoluto) se (palpite1 > numero_secreto) { diferenca1 = palpite1 - numero_secreto } senao { diferenca1 = numero_secreto - palpite1 } se (palpite2 > numero_secreto) { diferenca2 = palpite2 - numero_secreto } senao { diferenca2 = numero_secreto - palpite2 } // Decide o vencedor se (diferenca1 < diferenca2) { escreva("🏆 JOGADOR 1 VENCEU! 🏆\n") escreva("Diferença: ", diferenca1) } senao se (diferenca2 < diferenca1) { escreva("🏆 JOGADOR 2 VENCEU! 🏆\n") escreva("Diferença: ", diferenca2) } senao { escreva("🤝 EMPATE! Ambos acertaram igualmente! 🤝") } } }

Exercício 4: Cálculo IMC com Validação

Desafio: Calcule o IMC usando laço FAÇA-ENQUANTO para validar dados positivos!
programa { funcao inicio() { real peso, altura, imc escreva("=== CÁLCULO DO IMC ===\n") // Validação do peso usando FAÇA-ENQUANTO faca { escreva("Digite seu peso (kg): ") leia(peso) se (peso <= 0) { escreva("❌ Peso deve ser maior que 0!\n") } } enquanto (peso <= 0) // Validação da altura usando FAÇA-ENQUANTO faca { escreva("Digite sua altura (m): ") leia(altura) se (altura <= 0) { escreva("❌ Altura deve ser maior que 0!\n") } } enquanto (altura <= 0) // Cálculo do IMC imc = peso / (altura * altura) escreva("\n=== RESULTADO ===\n") escreva("Seu IMC: ", imc, "\n") // Classificação com SE SENÃO SE se (imc >= 16 e imc <= 16.9) { escreva("Classificação: Muito abaixo do peso") } senao se (imc >= 17 e imc <= 18.4) { escreva("Classificação: Abaixo do peso") } senao se (imc >= 18.5 e imc <= 24.9) { escreva("Classificação: Peso normal 🎉") } senao se (imc >= 25 e imc <= 29.9) { escreva("Classificação: Acima do peso") } senao se (imc >= 30 e imc <= 34.9) { escreva("Classificação: Obesidade Grau I") } senao se (imc >= 35 e imc <= 40) { escreva("Classificação: Obesidade Grau II") } senao se (imc > 40) { escreva("Classificação: Obesidade Grau III") } senao { escreva("Classificação: Fora da tabela de referência") } } }

Exercício 5: Tabuada Interativa

Desafio: Use o laço PARA para gerar a tabuada de um número entre 1 e 10!
programa { funcao inicio() { inteiro numero escreva("=== TABUADA INTERATIVA ===\n") faca { escreva("Digite um número de 1 a 10: ") leia(numero) se (numero < 1 ou numero > 10) { escreva("❌ Digite um valor entre 1 e 10!\n") } } enquanto (numero < 1 ou numero > 10) escreva("\n=== TABUADA DO ", numero, " ===\n") // Laço PARA para gerar a tabuada para (inteiro i = 1; i <= 10; i++) { escreva(numero, " x ", i, " = ", numero * i, "\n") } escreva("\nFim da tabuada!") } }
